What is the MutterButter Project?

The MutterButter Project is an overarching name for anything that I, Metacity, write or contribute to heavily enough. Other people might help me too as part of the project. You can help too - I love pull requests and issues!

What's here?

MutterKit

A flexible, modular IRC bot framework, using the equally flexible Mango core. Comes with a set of fun plugins, and facilities for you to write your own.

Language: Python 3
Project status: In development, not yet released

AuRepos

A complete, from scratch recoding of the famous IdleRPG, atop Mango. Can also be run as part of a full MutterKit bot.

Language: Python 3
Project status: In development, not yet released

Toolset

A set of scripts for the Eggdrop IRC bot, some complete, some incomplete, some original, some based on other scripts and cleaned up or expanded. Together they form a semi-cohesive whole.

Language: Tcl
Project status: Discontinued but still in use, may be revived and released
